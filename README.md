# shxx1f.github.io

这是 `shXX1F` 的 GitHub Pages 个人主页。

## 本地预览

直接打开 `index.html` 即可预览。也可以用任意静态服务器运行这个目录。

## 发布到 GitHub Pages

1. 在 GitHub 创建仓库：`shxx1f.github.io`
2. 把本目录里的文件提交到仓库根目录
3. 进入仓库 `Settings -> Pages`
4. 选择 `Deploy from a branch`
5. Branch 选择 `main`，目录选择 `/root`
6. 等待 GitHub Pages 完成部署

部署后地址通常是：

```text
https://shxx1f.github.io/
```

## 修改项目

页面会自动读取 `shXX1F` 的公开 GitHub 仓库。若想手动展示指定项目，可以编辑 `script.js` 里的 `fallbackProjects`。
